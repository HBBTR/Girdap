## Özellikler<br>
* Çoklu eşzamanlı indirme<br>
* Hız sınırlama ve trafik kontrolü<br>
* Duraklatma ve kaldığı yerden devam ettirme<br>
* Chrome/Edge tarayıcı eklentisi desteği<br>
* Pano takibi ile otomatik yakalama<br>
* Proxy ve güvenlik duvarı desteği<br>
* Zamanlayıcı ile planlı indirme<br>
* Sürükle-bırak ile sıralama<br>
* Detaylı istatistikler<br>
* API ile uzaktan kontrol
